# Apple-Native Setup Guide

## 1. Photos

Create an album named **Offers & Coupons**.

Use it for receipt photos connected to coupons, surveys, rebates, fuel points, warranties and return windows. Live Text in Photos provides on-device text recognition.

## 2. Reminders

Create a list named **Offers & Coupons**.

Suggested list settings:

- Show completed items when reviewing savings history.
- Keep notifications enabled.
- Allow location access when using store-arrival reminders.

Create these tags:

- `#Coupon`
- `#Survey`
- `#FuelPoints`
- `#Rebate`
- `#Return`
- `#Warranty`
- `#ReceiptRequired`
- `#Restaurant`
- `#Retail`

Create Smart Lists for:

- Due today
- Due within 3 days
- Due within 7 days
- Tagged ReceiptRequired
- Tagged Survey
- Tagged FuelPoints
- Tagged Return or Rebate

## 3. Calendar

Create a calendar named **Offers & Returns**.

Use Calendar only for items that deserve stronger visibility, such as:

- Return deadlines
- Rebate submission deadlines
- Warranty registration deadlines
- High-value coupons
- Fuel-point expiration

Recommended alerts:

- 7 days before
- 3 days before
- 1 day before
- On the expiration date

## 4. Notes

Create a folder named **Offers & Receipts**.

Use Notes when an offer has long terms, several codes, a serial number, a rebate checklist or warranty documentation. Pin the most active note when needed.

## 5. Files and iCloud Drive

Create this folder structure:

```text
Offers & Coupons/
├── Receipts/
├── Rebates/
├── Returns/
├── Warranties/
└── Exports/
```

Use PDF documents in Files for records that should not depend only on Photos or Reminders.

## 6. Action Button

After the **Save Receipt Offer** shortcut is built:

1. Open Settings.
2. Open Action Button.
3. Select Shortcut.
4. Choose **Save Receipt Offer**.

A press should launch receipt capture immediately.

## 7. Home Screen and Lock Screen

Recommended widgets:

- Reminders Smart List: Expiring Today
- Reminders Smart List: Expiring in 7 Days
- Calendar: Offers & Returns
- Shortcuts: Save Receipt Offer

## 8. Siri phrases

Recommended phrases:

- “Siri, save receipt offer.”
- “Siri, show offers expiring this week.”
- “Siri, show restaurant surveys.”
- “Siri, show receipt-required offers.”

## 9. Daily routine

1. Capture the receipt before leaving the store.
2. Confirm the expiration date.
3. Keep the original receipt when required.
4. Complete surveys promptly.
5. Review the 7-day Smart List once each morning.
6. Mark offers completed or redeemed to preserve history.
